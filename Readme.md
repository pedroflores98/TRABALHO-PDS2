# Chatbot

---------------------------------

Trabalho Prático Final da Disciplina de PDS2 - UFMG - 2019/2

Trata-se de chatbot que verifica se um individuo possui algum impedimento para doar sangue, e, caso não possua, o mesmo faz o seu cadastro.

Feito em C++
